auto 마진 이용 방법

가로너비 폭을 지정하고 수평 마진을 auto로 지정하기만 하면 된다.

.wrapper { width: 720px; margin: 0 auto; }

text-align 이용 방법 (hack)

부모 element에서 모든 자식 element를 가운데로 정렬시켰다가 다시 원래대로 좌측으로 정렬시킨다.

.wrapper2 { text-align: center; }
.wrapper2-inner { width: 720px; margin: 0 auto; text-align: left; }

포지셔닝과 음수 마진값 이용 방법

position: relative; left: 50%; 으로 wrapper3의 왼쪽 끝이 페이지의 중간으로 오게 하고

margin-left: -360px; 으로 wrapper3의 중간 점이 페이지 가운데로 오게 한다.

.wrapper3 { width: 720px; position: relative; left: 50%; margin-left: -360px; }
